### Installation:

To install and run this project in your system, you need to set up MeteorJS and install all related dependencies.

#### **Install Node.js**

Node.js is a dependency required to install MeteorJS into your system.
- Download Node.js from: https://nodejs.org/
- Install the latest version according to your operating system.
- After installation, open your terminal and type these commands:
    ```bash
    node -v
    npm -v
    ```
- If both commands return version numbers, then installation is successful.

#### **Install MeteorJS**

This web app is built using the MeteorJS Framework. It is required for the app to function.
- Open your terminal and type in the following command:
    ```bash
    npn install -g meteor
    ```
- This will download and install Meteor on your system.
- Restart your terminal, and verify the installation by running:
    ```bash
    meteor --version
    ```

#### **Install the Project**

You may proceed to clone the repository, or download the project folder.
- Use your terminal or console to navigate to the folder.
    ```bash
    cd firstattempt2026_macrohon
    ```
- Once inside the project folder, initialize the server by typing:
    ```bash
    meteor
    ```
- You may view the application at `http://localhost:3000`.
